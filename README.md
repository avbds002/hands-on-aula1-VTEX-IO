# Projeto Hands-on HTML CSS GrowDev

Código HTML & CSS do desafio proposto na aula síncrona de HTML e CSS.

Link do design proposto para o desafio no figma: https://www.figma.com/make/RZ93PPFasAkeS1uYA8Vs0x/Educational-Wireframe-Design?p=f&fullscreen=1

A atividade proposta visa avaliar o quanto o aluno sabe sobre os fundamentos de HTML & CSS e também funciona como uma revisão dos conceitos abordados durante as aulas do módulo de HTML & CSS.
